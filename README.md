## 🚀 使用指南

### 一. 下载图标
在仓库文件夹中找到你需要的图标文件（`.png` 或 `.jpg`），下载并保存至手机相册。

### 二. 在 App 中替换
1. 打开 **ASUS Router App**，点击底部的“**设备**”选项卡。
2. 选中你想要修改的设备（例如你的 MacBook Pro 或 Apple TV）。
3. 点击当前的默认设备图标。
4. 在弹出的菜单中选择 **“从相册选择”**。
5. 选中下载好的图标，调整裁剪区域后保存。
